## Environment Variables

- `DISCORD_WEBHOOK`: Your Discord Webhook URL.
- `KOFI_TOKEN`: Ko-fi Verification Token. Used to verify event is sent by Ko-fi.
- `MONGO_URI`: MongoDB URL.
